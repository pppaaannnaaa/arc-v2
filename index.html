    ARC v2.1 ‚Äî AI Readability Convention¶

    A Specification for Machine-Readable Web Interfaces

    ARC (AI Readability Convention) is a semantic markup system that uses lightweight, inline HTML attributes (data-ai-*) to make interfaces interpretable by AI systems without needing external metadata, JSON-LD, or backend schemas.

    This repository contains the complete ARC v2.1 specification, fully expanded and structured into sections and appendices.
