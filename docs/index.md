# ARC v2.1 â€” AI Readability Convention  
*A Universal Inline Semantic System for AI-Interpretable Interfaces*

ARC (AI Readability Convention) provides a lightweight, declarative way to make websites and applications **machine-readable** using inline HTML attributes.

The goal is simple:

> **Give AI systems a way to understand web interfaces without external schemas, fragile parsing, or guesswork.**

ARC attributes (`data-ai-*`) describe:
- structure  
- content  
- relationships  
- behavior  
- intent  
- context  
- dynamic interactions  

All directly within the HTML itself.
